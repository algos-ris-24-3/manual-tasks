## Задание

| Производительность(p) |  8  |  4  |  3  |  2  |

## Решение
Находим длину расписания, разделив всю длительность на производительность

t=(54+37+35+25+19+17+17)/(8+4+3+2)=12

### Вычисляем время события с помощью уравнений, используя данные из таблиц

A:B=54-8t=37-4t => 4t=17 => t=4.25

B:C=37-4t=35-3t => **t=2**

C:D=35-3t=25-2t => t=10

D:E=25-2t=19 => 2t=6 => t=3

Далее сравнивать задачи нельзя, так как исполнителей на задачу E не назначено. Таким же образом и в дальнейших вычислениях мы не будем считать задачи без исполнителей.

### Заносим данные в диаграмму Ганта
Так как задачи по времени разные, то на одну задачу распределяем по одному исполнителю. На самую большую по длительности задачу распределяем самого производительного исполнителя и так далее, пока не закончатся исполнители.

### Рассчитаем время нового события исходя из новых длительностей
Так как задачи C и D одинаковой длительности и обе выполняются двумя исполнителями, то вычисляем их t при помощи сложения производительностей нужного кол-ва исполнителей (в нашем случае - 2) и деления этого значения на кол-во задач. Выбираем наименьшее t.

A:BC=38-8t=29-7t/2 => 76-16t=58-7t => 18=9t => t=2

BC:D=29-7t/2=21-2t => 58-7t=42-4t => 16=3t => t=16/3=5.(3)

D:E=21-2t=19 => 2=2t => **t=1**

Так как длительности для задач В и С одинаковые, то мы выбираем максимальное количество исполнителей для этих задач. Так как задач 2, то максимальное количество исполнителей - 2. Одна и та же задача также не может быть одновремеенно у двух исполнителей, учтём это при заполнении.
