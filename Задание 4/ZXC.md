### Отбор задач, где второй этап ≥ первого

|    | A  | **B** | C | D | **E** | **F** | **G** | **H** | I |
| -- | -- | ----- | - | - | ----- | ----- | ----- | ----- | - |

### Порядок по правилу Джонсона

#### 1. В начало — задачи с t₂ ≥ t₁ (по возрастанию t₁):

**B, G, E, F, H**

#### 2. В конец — задачи с t₁ > t₂ (по убыванию t₂):

**A, D, I, C**

#### Итоговая последовательность:

**B → G → E → F → H → A → D → I → C**
